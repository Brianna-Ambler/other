About This Workshop

Learn how to upgrade and migrate your database to the latest version of Oracle AI Database. In this workshop, you learn how to use AutoUpgrade to make upgrade and non-CDB to PDB migrations much easier. Further, you can use our Performance Stability Prescription to help you to avoid the biggest fear of a DBA: bad execution plans after the upgrade. Using tools like SQL Performance Analyzer and SQL Plan Management, you can ensure performance stability after any change — including upgrades and migrations. You will also try migrating a database to a PDB using various techniques. Finally, you learn how to restore after a failed upgrade and downgrade a PDB.
